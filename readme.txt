Добавление ребра:
add (первая верщина, вторая вершина, вес)
Пример ввода:
add (1, 3, 5)
Пример вывода:
(1, 4, 5) (1, 3, 5) (3, 1, 5) (4, 1, 5)

Удаление ребра:
remove (первая верщина, вторая вершина)
Пример:
remove (1, 3)
Пример вывода:
(1, 4, 5) (4, 1, 5)

Обновление веса:
changeEdgeWeight (первая верщина, вторая вершина, новый вес)
Пример ввода:
changeEdgeWeight (1, 3, 7)
Привер вывода:
(1, 4, 5) (1, 3, 7) (3, 1, 7) (4, 1, 5)


Поиск кратчайшего пути:
findShortestPath (начальная верщина, конечная вершина)
Пример ввода:
findShortestPath (3, 4)
Пример вывода:  сначала выводится кратчайший путь, после итоговый вес
(3, 1, 4, 12)
